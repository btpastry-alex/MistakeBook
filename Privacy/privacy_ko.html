개인정보 처리방침

본 방침은 MistakeBook이 각 플랫폼에서 데이터를 어떻게 처리하는지, 어떤 권한을 사용하는지, 그리고 사용자가 자신의 데이터와 권한을 어떻게 관리할 수 있는지를 설명합니다.

1. 데이터 수집

iOS 버전

iOS 버전의 MistakeBook은 어떠한 사용자 데이터도 수집하지 않습니다. 이름, 이메일 주소, 전화번호, 주소와 같은 개인 식별 정보를 수집·업로드·저장하지 않으며, 탐색, 탭, 상호작용 기록과 같은 사용 데이터도 분석이나 광고 목적으로 수집하지 않습니다.

Android 버전

Android 버전의 MistakeBook 자체는 사용자가 생성한 오답 콘텐츠, 메모, 이미지, 분류 데이터를 직접 수집·업로드·저장하지 않습니다. 다만 Android 버전에는 제3자 광고 서비스인 Google AdMob이 포함되어 있으며, 광고, 분석, 사기 방지 또는 법규 준수와 관련된 다음 데이터가 처리될 수 있습니다.

  • 대략적인 위치
  • 앱 상호작용
  • 진단 정보
  • 기기 ID 또는 기타 ID

실제 공개 내용은 Google Play 스토어의 Data safety 항목을 따릅니다.

2. 사용자와의 연결 여부

iOS 버전

iOS 버전은 데이터를 수집하지 않으므로 사용자 신원과 연결되는 데이터가 없습니다. 계정 로그인, 회원 시스템, 클라우드 동기화를 제공하지 않으며 사용자 프로필도 생성하지 않습니다.

Android 버전

Android 버전의 MistakeBook 자체도 계정 로그인, 회원 시스템, 클라우드 동기화를 제공하지 않으며 MistakeBook 사용자 프로필을 생성하지 않습니다. 다만 Android 버전에서 사용하는 제3자 광고 서비스는 광고 제공, 광고 측정, 사기 방지 또는 법규 준수를 위해 일부 데이터를 처리할 수 있습니다.

3. 추적 및 광고

iOS 버전

iOS 버전의 MistakeBook은 추적에 사용되지 않습니다. 앱 또는 웹사이트 간 사용자 활동을 추적하지 않으며, 광고 제공이나 측정을 위해 제3자와 데이터를 공유하지 않습니다.

Android 버전

Android 버전의 MistakeBook은 Google AdMob에서 제공하는 광고를 표시합니다. 따라서 Android 버전에서는 제3자 광고 서비스가 일부 데이터를 다음 목적에 따라 처리할 수 있습니다.

  • 광고 또는 마케팅
  • 데이터 분석
  • 사기 방지, 보안 및 법규 준수

사용자의 지역에 따라 관련 개인정보 보호 법규가 적용되는 경우, 제3자 광고 서비스가 동의 관리나 개인정보 선택 기능을 제공할 수 있습니다.

4. 기기 내 처리 및 저장

본 앱에서 생성하거나 가져온 콘텐츠(문제/정답 이미지, 주석, 메모, 분류 정보, 숙련도, 복습 기록 등)는 기기 내에서 처리되며, 사용자의 기기 로컬 영역(앱 샌드박스 / 앱 전용 저장소)에 저장됩니다. MistakeBook 자체는 사용자의 오답 콘텐츠를 개발자의 자체 서버로 자동 업로드하지 않습니다.

다만 Android 버전에서는 제3자 광고 서비스가 통합되어 있으므로, 광고 관련 SDK가 일부 기기 및 사용 데이터를 처리할 수 있습니다. 이는 오답 콘텐츠 자체가 업로드된다는 의미는 아닙니다.

5. 권한

본 앱은 필요 시 다음 시스템 권한을 요청하며, 관련 기능을 직접 사용할 때만 접근합니다.

  • 카메라: 문제/정답 이미지를 촬영하여 앱에 추가하기 위해 사용합니다.
  • 사진 / 미디어: 사진 라이브러리에서 이미지를 선택하여 앱에 추가하기 위해 사용합니다.
  • 클립보드 / Pasteboard: “붙여넣기” 기능 사용 시 복사된 내용을 읽기 위해 사용하며, 백그라운드에서 지속적으로 읽지 않습니다.
  • 파일 / 저장소 접근: 시스템 파일 관리자에서 사용자가 선택한 파일을 가져오기/내보내기 위해 사용합니다. 선택한 파일만 접근하며 다른 파일은 스캔하지 않습니다.

이러한 권한은 시스템 설정(iOS / Android)에서 언제든지 관리하거나 철회할 수 있으며, 철회 시 일부 기능이 제한될 수 있습니다.

6. 가져오기 / 내보내기 / 공유

파일은 사용자가 내보내기 또는 공유 기능을 명시적으로 실행할 때만 생성되며, 이후 시스템의 저장 또는 공유 기능을 통해 처리됩니다. MistakeBook은 사용자 파일을 백그라운드에서 개발자의 자체 서버로 자동 업로드하지 않으며, 자체 클라우드 동기화 기능도 제공하지 않습니다.

사용자가 내보낸 파일을 제3자 서비스(예: 클라우드 드라이브, 메신저 앱)로 공유하는 경우, 이후의 데이터 처리는 해당 서비스의 정책과 약관에 따릅니다.

7. 데이터 삭제

앱 내에서 오답 콘텐츠를 삭제할 수 있습니다. 앱을 삭제하면 기기 내에 저장된 MistakeBook의 로컬 데이터 (앱 샌드박스 / 전용 저장소)가 제거됩니다. 외부 위치 또는 제3자 서비스에 내보낸 백업은 사용자가 직접 삭제해야 합니다.

8. 변경

향후 앱 버전에서 데이터 수집이 추가되거나, 제3자 광고 또는 분석 서비스가 변경되거나, 클라우드 동기화 기능이 도입되는 경우 본 방침은 업데이트되며 앱 또는 웹사이트에 안내됩니다.
