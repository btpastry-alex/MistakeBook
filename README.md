# MistakeBook (iOS)

MistakeBook is a mobile “mistake notebook” app built around spaced repetition inspired by the Ebbinghaus Forgetting Curve. Capture/import question images, crop precisely, annotate on top of images, and review with reminders—everything stored locally on-device.

## Key Features
- **Capture & Import**: Camera / Photo Library / Paste images into your mistake set
- **Precise Crop**: Quickly isolate the exact question area you need
- **Annotate & Mask**: Draw (pen/eraser), adjust colors & stroke width, undo/redo, and optionally mask/whiten-out
- **Review Workflow**: “Today” and “Upcoming” views with reminder scheduling  
  - Rate familiarity after review; next review date is auto-planned  
  - Items rated **5★ (mastered)** move to **Completed** and stop scheduling
- **Export & Share**
  - Export questions/answers/review records to **A4 PDF**
  - Export/Import via a compressed archive to share mistake sets
- **Stats**: Visualize bottlenecks and learning blind spots
- **Worksheet Builder**: Generate printable worksheets in **PDF** or editable **ODF**

## Tech Stack
- **SwiftUI**
- **SwiftData / Core Data** (local storage)
- PDF export

# MistakeBook（錯題本）(iOS)

MistakeBook（錯題本）は、エビングハウスの忘却曲線に着想を得た「間隔反復（Spaced Repetition）」を軸にした学習支援アプリです。問題画像を取り込み→トリミング→画像上に注釈→リマインドで復習、すべて端末内に保存します。

## 主な機能
- **取り込み**：カメラ／写真ライブラリ／貼り付けから画像を追加
- **精密トリミング**：必要な範囲を素早く切り出し
- **注釈・マスキング**：ペン／消しゴム、色・太さ調整、Undo/Redo、必要に応じて隠し（白塗り）処理
- **復習フロー**：「今日」「近日（即將）」で復習・通知  
  - 復習後に習熟度を評価 → 次回復習日を自動スケジュール  
  - **★5（完全習得）**は**完了**へ移動し、以後は通知対象外
- **エクスポート／共有**
  - 問題・解答・復習記録を **A4 PDF** で出力
  - 圧縮ファイルで Export/Import し、セットを共有可能
- **統計**：弱点や学習の詰まりを可視化
- **出題（出卷）**：選択した錯題から **PDF** / 編集可能な **ODF** を生成（手書き練習用）

## 技術スタック
- **SwiftUI**
- **SwiftData / Core Data**（ローカル保存）
- PDF 出力

# MistakeBook（错题本）(iOS)

MistakeBook（错题本）是一款以“间隔重复”学习方法为核心的错题整理 App（灵感来自艾宾浩斯遗忘曲线）。导入题目图片→精准裁切→在图上标注→按提醒复习；所有数据都保存在本机。

## 核心功能
- **导入方式**：相机 / 相册 / 粘贴导入题目图片
- **精准裁切**：快速截取需要的题目范围
- **标注与遮蔽**：画笔/橡皮擦、颜色与粗细、Undo/Redo，并可按需遮蔽/抹白
- **复习流程**：「今日」「即将」视图 + 提醒机制  
  - 复习后自评熟悉度 → 自动安排下次复习时间  
  - **5★（完全掌握）**自动归入**已完成**，不再进入复习排程
- **导出与分享**
  - 题目/答案/复习记录导出为 **A4 PDF**
  - 压缩包 Export/Import，便于分享错题集
- **统计**：用数据发现学习瓶颈与盲点
- **出卷**：勾选错题生成可打印 **PDF** 或可编辑 **ODF** 练习卷

## 技术栈
- **SwiftUI**
- **SwiftData / Core Data**（本地存储）
- PDF 导出
